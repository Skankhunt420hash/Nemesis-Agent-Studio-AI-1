# OpenClaw Agent Studio

Agenten bauen, füttern, trainieren und per Telegram/OpenClaw automatisieren.

## Enthält
- Login/Register
- Workspaces
- Agent-Builder
- Wissen / Feed / Training
- Telegram- und OpenClaw-Automation
- Public Chat
- Hosted Export unter `/generated/<slug>/`

## Start

```bash
npm install
cp .env.example .env
npm start
```

## Test

```bash
npm test
```
